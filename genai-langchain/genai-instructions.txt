To run the genai related langchain code in this repo, follow these steps:

1. Install uv (pthon env/package manager): 
        pip install uv (OR)
        (windows) irm https://astral.sh/uv/install.ps1 | iex
2. Install python 3.12 (dont use 3.13/3.14 right now): uv python install 3.12
3. Create a virtual environment named .venv-langchain with python 3.12: uv venv .venv-langchain --python 3.12
4. Activate the virtual environment: uv venv .venv-langchain --activate
5. Install required packages: uv pip install -r requirements.txt
